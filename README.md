# DLSU Academic Support Hub

Vercel-ready Next.js website for the Student Success Center Academic Support Hub.

## Local setup

1. Install Node.js 22 or later.
2. Run `npm install`.
3. Run `npm run dev`.
4. Open `http://localhost:3000`.

## Deploying to Vercel

Upload this project to a GitHub repository, import the repository in Vercel, and keep the detected Next.js defaults. No environment variables are required.
