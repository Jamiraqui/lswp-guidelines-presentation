# LSWP Interactive Presentation

## Fastest publishing option: Vercel

1. Sign in at https://vercel.com.
2. Create a new project and choose **Upload** or import this extracted folder from GitHub.
3. Keep the detected framework settings. The project uses Next.js.
4. Click **Deploy**.
5. When deployment finishes, use the provided `vercel.app` address for the board presentation.

## Run it on a computer

Install Node.js 22 or later, then open a terminal in this folder and run:

```bash
npm install
npm run dev
```

Open the local address shown in the terminal. Use the **Present** button in the upper-right corner for the board session.

## Main files

- `app/page.tsx`: presentation content and interactions
- `app/globals.css`: visual design and responsive layout
